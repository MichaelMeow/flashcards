JavaScript (JS)

JavaScript is the programming language of HTML and the Web.

Operators

Symbols that are used to perform an operation. Most often these operations are arithmetic.

Variables

JavaScript variables are containers for storing data values.

Variable naming conventions

Use camelCase for identifier names.

Functions

A set of statements that performs a task or calculates a value.

Methods

Actions that can be performed on objects

Arguments

Function arguments are the real values passed to (and received by) the function.

Parameters

Function parameters are the names listed in the function definition.

Return

When JavaScript reaches a return statement, the function will stop executing.

Chaining methods

Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Strings

A series of characters used for storing and manipulating text.

Booleans

A primitive data type that represents one of two values: true or false.

Undefined

A primitive data type that is used when a variable has not been assigned a value.

NaN

A value in the number data type representing Not-A-Number.

Escape

The backslash (\) escape character turns special characters into string characters.

Alert

displays an alert box with a specified message and an OK button.

Comments

Any text between // and the end of the line will be ignored by JavaScript (will not be executed).

jQuery

A JavaScript library that allows web developers to add extra functionality to their websites.

Attributes

Provides additional information about an element.
